# Seraphis (in-progress draft)

Seraphis is a privacy-focused transaction protocol for p2p electronic cash systems (e.g. cryptocurrencies).

Browsers with built-in PDF readers (e.g. recent versions of Firefox) can display the most recent PDF [with this link](https://raw.githubusercontent.com/UkoeHB/Seraphis/master/Seraphis-0-0-4.pdf). Note that non-supporting browsers may automatically download the PDF if you click the link.
